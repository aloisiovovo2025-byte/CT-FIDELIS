# CT Fidelis Landing Page - PRD

## Original Problem Statement
Landing page profissional para captação de alunos do centro de treinamento de Taekwondo CT Fidelis, focada em conversão através de aula gratuita via WhatsApp.

## User Choices
- Tema escuro (preto com acentos vermelho)
- Tipografia: **ETHNOCENTRIC** oficial (títulos) + Manrope (corpo)
- Animações profissionais **slide-in** ao scroll
- Frontend estático apenas (sem backend)
- Logo, fotos e ícone WhatsApp fornecidos pelo usuário
